## 실시간 혼잡도 분석기
 신갈고등학교는 약 880명 가량의 학생들이 점심시간 급식을 먹기위해 급식실에 줄을 서지만, 
 급식실의 수직적 구조탓에 학생회의 급식지도 학생들이 어려움을 겪는 문제가 존재했습니다. 
 때문에 이를 해결하기 위해 급식실 입구에 카메라를 설치하고, 급식실 입장 대기자의 수를 수치화하는 
 인공지능 기반 실시간 혼잡도 분석 프로그램을 개발하였습니다.
